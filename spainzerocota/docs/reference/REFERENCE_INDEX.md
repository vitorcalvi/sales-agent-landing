## 🔑 Key Documents Needed

1. **Identity**
   - Italian Passport
   - NIE (foreigner ID number)
   - NIE card copies

2. **Residency**
   - Empadronamiento certificate (< 3 months old for grants)
   - Rental contract

3. **Business**
   - Modelo 036 (tax registration)
   - RETA certificate (social security)

4. **Banking**
   - Spanish IBAN account
   - Bank statement proof
